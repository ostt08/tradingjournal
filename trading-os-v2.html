Integrations & Automations

TradingView → Webhook
PRIORITY 1
Goal: Alert fires in TradingView → hits your system
1. TradingView Pro ($15/mo) → Create alert on EMA cross
2. Set webhook URL to your N8N instance
3. Payload: {"pair":"{{ticker}}","signal":"{{strategy.order.action}}"}
4. N8N receives → Claude validates R:R → notifies you
⚙️
N8N Automation
PRIORITY 2
Self-hosted on Railway (free tier):
Flow: TradingView Webhook → N8N → Claude API node (validate signal) → Telegram → You approve → Log to Google Sheet
Install: npm install n8n -g && n8n start
Key nodes: Webhook · HTTP Request · Telegram · Google Sheets · Code node (JS)

RECOMMENDED

2. N8N Telegram node → send structured message
